Ejemplo de Applet con Java

Un applet conforme a lo que encontramos en las paginas de oracle" Es una aplicacion hecha en Java desde un documento Html y se ejecuta dentro de un navegador". Es necesario recordar que el navegador debe tener instalado un plugin de la Maquina virtual de java (JVM)

Maquina virtual de java es el programa ese el programa que se encarga del bytecode genrdo por compilar el java.

Para entender la forma en que se manejan los applets vamos a ver un ejemplo que representa el clasico Hola mundo

<>Como puede observar la unica variable que utliliza este applet es un g representa un objeto de la clase graphics sobre la cual vamos a dibujar de texto que resulta ejecutar el applet

Para ejecutarlo es necario mandarlo a llamar desde una pagina HTML utilizando la etiqueta >applet< o la etiqueta >objetct<.En el primer caso el archivo .class se declara el atributo codeen el segundo se declara un elemento param donde sus atributos name sera icode y value sera el applet. Cabe mencionar que en el segundo caso es necesario declarar elementos param adicionales y en elemento object declarar algunos parametros como classid,codebase y type. En ambos casos es necesario declarar el tamaño de la ventana del applet con los parametros del width y height.

La salida del programa sera: Hola mundo¡¡
